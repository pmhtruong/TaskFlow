# TaskFlow

A modern task management web app focused on simplicity, productivity, and clean UI.

## Features
- Create new tasks
- Mark tasks as completed
- Delete tasks
- Editable project title
- Persistent storage with LocalStorage

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Current Status
In Development 
- Todo module completed

## Future Plans
- Task filtering
- Calendar integration
- Task detail modal
- Dark mode
- Backend integration

## Project Goal
This project is built to practice frontend development, UI design, and real-world application logic.
